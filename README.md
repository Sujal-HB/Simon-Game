# Simon Game 🎮

A simple, interactive Simon memory game. Test your memory by repeating an ever-increasing sequence of color!

---

## 🚀 How to Play
1. **Start the Game**: Press any key to begin.
2. **Watch the Sequence**: The game will flash a button.
3. **Repeat the Pattern**: Click the buttons in the exact order they were shown.
4. **Level Up**: Every successful round adds a new step to the sequence. If you click the wrong button, it's Game Over!

## 🛠️ Built With
* **HTML** - Structure and layout
* **CSS** - Styling, grid positioning, and animations
* **JavaScript** - Game logic, DOM manipulation, and audio integration
